Summaries

One sentence

Attention Is All You Need introduces the Transformer, a sequence model that relies entirely on self-attention and removes recurrence and convolutions.

Three sentences
  1. The Transformer uses multi-head scaled dot-product self-attention to model token relationships in parallel.
  2. It removes recurrence and convolutions, enabling much faster training and better long-range modelling on translation and other tasks.
  3. On WMT 2014 En-De and En-Fr translation the Transformer surpasses prior encoder-decoder baselines while using a fraction of the training compute.
Ten sentences
  1. The paper proposes the Transformer, a new sequence transduction architecture.
  2. The core building block is multi-head scaled dot-product self-attention.
  3. Self-attention allows every position to attend to every other position in O(1) sequential steps.
  4. Encoder and decoder are stacks of identical layers with attention and position-wise feed-forward sub-layers.
  5. Sinusoidal positional encodings inject order information.
  6. Training uses Adam with a warmup-then-decay learning rate schedule.
  7. Label smoothing is applied to the output projection.
  8. On WMT 2014 En-De the base model reaches 27.5 BLEU and the big model 28.4 BLEU.
  9. On WMT 2014 En-Fr the big model reaches 41.8 BLEU, a single-model state of the art at the time.
  10. The architecture has since become the foundation of most large language models.

Pass 2 — Understanding

Main ideas

  1. Replace recurrence/convolutions with self-attention.
  2. Use multiple attention heads in parallel.
  3. Use sinusoidal positional encodings.
  4. Train with Adam + warmup + label smoothing.

Pass 3 — Reconstruction

Method reconstruction

  1. Tokenize input and embed with d_model=512.
  2. Add sinusoidal positional encoding.
  3. Apply N=6 encoder layers; each layer is multi-head self-attention + position-wise FFN with residual + LayerNorm.
  4. Decoder is similar with masked self-attention and encoder-decoder attention.
  5. Linear + softmax projection to vocabulary, label smoothing 0.1.
  6. Adam with warmup-then-decay LR, dropout 0.1.

Critical review highlights

  • Quadratic memory in sequence length is the main scaling concern.
  • Sinusoidal positions were later replaced by RoPE / ALiBi in many follow-ups.
  • Label smoothing slightly hurts perplexity but improves BLEU and calibration.
